# Directory Structure Summary

This repository adds a Claude Code custom command suite and a feature-development docs area.

## Top-level layout
- .claude/commands/ (custom slash commands)
- docs/claude-code/ (Claude Code reference docs for this repo)
- docs/feature-dev/ (generated feature workflow docs)

## Custom slash commands
- /001: Create the "why needed" doc
- /002: Investigate related code/docs
- /003: Generate the feature document
- /004: Implement tests
- /005: Implement code
- /006: Validate behavior
- /007: Create pull request

## File naming conventions (in docs/feature-dev/)
- YYYYMMDD_{機能タイトル}.md
- YYYYMMDD_{機能タイトル}_調査結果.md
- YYYYMMDD_{機能タイトル}_ドキュメント.md
