# 企業情報をわかりやすく表示させる

- **Date:** 2026-02-22

## Why Needed

チャート情報の上に企業情報や株価、更新日時を表示させて一目でわかるようにする。

現在の実装では会社情報テキスト（銘柄名・現在値・前日比・出来高・時価総額・取得日時）はチャートの**下部**に小さく表示されているが、視認性が低い。チャートを見た瞬間に「どの銘柄・いくら・いつの情報か」が直感的にわかるよう、チャートの**上部（タイトル領域）** に企業情報を明示的に配置する。

---

## Scope

### In-scope
- チャートタイトル行を「銘柄コード + 期間」から「銘柄名・現在値・前日比・更新日時」を含む情報豊富なタイトルへ変更
- `plot_intraday()` のタイトル表示を更新
- `plot_intraday_live()` のタイトル表示を更新（自動更新のたびに最新情報へ書き換え）
- 既存の下部テキスト (`fig.text`) の扱い（削除 or 残存）の決定

### Out-of-scope
- CUI（コンソール）側の表示フォーマット変更（`print_stock_info` は対象外）
- チャート以外のウィンドウや別パネルの追加
- WebSocket・リアルタイムティッカー表示
- 複数銘柄の同時表示

---

## Success Criteria

- `plot_intraday()` 実行時、チャートウィンドウ上部（タイトル部）に銘柄名・現在値・前日比・更新日時が表示される
- `plot_intraday_live()` 実行時、自動更新のたびにタイトル部の情報が最新データで更新される
- 既存の 70 件のテストが引き続き PASS する
- 追加したテストが PASS する

---

## Risks/Unknowns

- mplfinance の `title=` パラメータは文字列のみ受け付けるため、動的な更新には `fig.suptitle()` / `axes[0].set_title()` への切り替えが必要になる可能性がある
- `plot_intraday_live()` の `animate` コールバック内でタイトルを更新する際、`ax.clear()` 後にタイトルが消えるかどうかの挙動確認が必要
- 長い銘柄名でタイトルが折り返しや省略になる場合のレイアウト崩れリスク
- 日本語フォントが解決されない環境での文字化けリスク（`_resolve_japanese_font()` の挙動依存）
