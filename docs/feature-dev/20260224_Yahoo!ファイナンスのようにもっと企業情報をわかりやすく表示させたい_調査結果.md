## Approach

### matplotlib `fig.text()` の特性確認

- `fig.text(x, y, s, fontsize=N, color='#rrggbb', fontweight='bold', va='top', ha='left')` でフィギュア座標系（0–1）にテキストを配置できる
- 戻り値は `matplotlib.text.Text` オブジェクト
- `text_obj.set_text(s)` でテキスト更新、`text_obj.set_color(c)` で色更新が可能
- **`ax.clear()` はフィギュアレベルのアーティストに影響しない** → `animate()` 内で `ax1.clear()` / `ax2.clear()` を呼んでも `fig.text()` オブジェクトは残る（前フィーチャーと同じ理由）

### 採用するレイアウト設計

`fig.subplots_adjust(top=0.82)` で上部に2行分のスペースを確保し、`fig.text()` を計4オブジェクト配置する。

```
y=0.975  [銘柄名 (コード)  bold 13pt]  [¥現在値  bold 16pt]  [前日比  12pt green/red]
y=0.930  [出来高: xxx  時価総額: xxx  取得: YYYY-MM-DD HH:MM  9pt gray]
```

#### 各テキストオブジェクト

| 変数名 | x | y | 内容 | fontsize | color |
|---|---|---|---|---|---|
| `name_txt` | 0.01 | 0.975 | `"{info.name} ({info.ticker})"` | 13 | `#333333` |
| `price_txt` | 0.42 | 0.975 | `"¥{current_price:,.0f}"` | 16 | `#333333` |
| `change_txt` | 0.58 | 0.975 | `"{sign}{diff:,.0f} ({sign}{pct:.2f}%)"` | 12 | `#22ab94`（正）/ `#f7525f`（負） |
| `meta_txt` | 0.01 | 0.930 | `"出来高: {vol}  時価総額: {cap}  取得: {ts}"` | 9 | `#666666` |

#### ヘルパー関数の追加

```python
def _build_info_parts(df: pd.DataFrame, info: StockInfo) -> dict:
    """チャート上部の各テキストパーツを辞書で返す。

    Returns:
        {
            "name": str,         # 銘柄名 + コード
            "price": str,        # 現在値文字列
            "change": str,       # 前日比文字列
            "change_color": str, # 色コード（正=緑, 負=赤）
            "meta": str,         # 出来高・時価総額・取得日時
        }
    """
    diff = info.current_price - info.previous_close
    pct  = (diff / info.previous_close * 100) if info.previous_close else 0.0
    sign = "+" if diff >= 0 else ""
    ts   = df.index[-1].strftime("%Y-%m-%d %H:%M")
    return {
        "name":         f"{info.name} ({info.ticker})",
        "price":        f"¥{info.current_price:,.0f}",
        "change":       f"{sign}{diff:,.0f} ({sign}{pct:.2f}%)",
        "change_color": "#22ab94" if diff >= 0 else "#f7525f",
        "meta":         f"出来高: {info.volume:,}  時価総額: {_format_market_cap(info.market_cap)}  取得: {ts}",
    }
```

`_build_info_text()` は `TestBuildInfoText` の4件テストが依存しているため**削除せず残す**。

## Risks/Constraints Discovered

1. **`TestPlotIntradayInfoText` 7件が大きく変化**: `fig.suptitle` → `fig.text` の複数呼び出しに変わるため、`call_args[0][0]` の単一アクセスではなく `call_args_list` を走査するか、呼び出し回数 (`call_count`) を確認するパターンに変更が必要
2. **`test_animate_updates_info_text` 変更**: `suptitle.return_value.set_text` → `_change_txt.set_text`・`_change_txt.set_color` の確認に変更が必要。`fig.text` の呼び出し順序に依存した検証になる
3. **x座標のハードコード**: `figsize=(14, 7)` を前提とした x 座標を使うため、figsize 変更時にレイアウトがずれる可能性がある（現時点では figsize 変更の要件がないため許容）
4. **`_name_txt` は `animate()` で更新しない**: 銘柄名は変わらないため省略できるが、テストで「`set_text` が呼ばれない」ことを確認する必要がある

## Open Questions

1. `change_color` の緑・赤の具体的なカラーコードは `#22ab94` / `#f7525f`（Yahoo!ファイナンス準拠）で良いか、それとも matplotlib 標準の `green` / `red` で十分か → ドキュメントで `#22ab94` / `#f7525f` を採用する
2. 前日比ゼロ（`diff == 0`）の場合の色は緑・赤どちらにするか → ゼロ以上（`diff >= 0`）は緑として扱う（変化なし＝マイナスではないため）
3. `_build_info_text()` を使う箇所がなくなったら削除してよいか → テストが依存しているため**残す**
