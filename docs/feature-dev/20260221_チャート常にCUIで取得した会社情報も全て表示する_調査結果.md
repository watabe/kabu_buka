## Findings Summary

### `returnfig=True` の動作確認（実環境）

```python
fig, axes = mpf.plot(df, type='candle', style=style, volume=True,
                     figsize=(14, 7), returnfig=True)
# fig: matplotlib.figure.Figure
# axes: 4要素リスト（price×2 + volume×2、パネル数×2軸構成）
```

- `returnfig=True` を指定すると `(fig, axes)` タプルを返し、`mpf.show()` を呼ぶまで表示しない
- `fig.text(x, y, text, fontsize=...)` で figure 座標（0〜1）に任意テキストを追加可能
- `fig.subplots_adjust(bottom=0.12)` で下余白を確保してからテキストを配置する手法が有効

### 取得日時の導出

DataFrameのインデックス（`df.index[-1]`）が直近バーの `Timestamp` （Asia/Tokyo）であるため、`df.index[-1].strftime('%Y-%m-%d %H:%M')` で「何日何時現在」を文字列化できる。

### 表示する情報の候補

`StockInfo` から以下が取得可能：

| フィールド | 内容 |
|---|---|
| `info.name` | 銘柄名 |
| `info.ticker` | ティッカー（例: 7203.T） |
| `info.current_price` | 現在値 |
| `info.previous_close` | 前日終値（前日比計算に使用） |
| `info.open_price` | 始値 |
| `info.day_high` / `info.day_low` | 高値 / 安値 |
| `info.volume` | 出来高 |
| `info.market_cap` | 時価総額 |

取得日時は `df.index[-1]`（intraday DataFrame の最終行タイムスタンプ）から取る。

## Risks / Constraints Discovered

- **`returnfig=True` 追加で `mpf.show()` の明示呼び出しが必須。** 既存コードは暗黙的に表示していたため、`mpf.show()` を追加しないとウィンドウが開かなくなる。
- **`subplots_adjust` のデフォルト値と干渉の可能性。** `mplfinance` が内部で `subplots_adjust` を設定している場合、後から `fig.subplots_adjust(bottom=...)` を呼ぶと上書きになる。実際には後呼び出しが優先されるが、`tight_layout` との同時使用は避ける。
- **テキスト幅の溢れ。** `fig.text` は折り返し処理を自動で行わない。1行に収まらない場合は2行に分割するか、`fontsize` を小さくする必要がある。14×7 inch の figure では fontsize=9 程度が適切（実環境確認済み）。
- **シグネチャ変更により `tests/test_chart.py` の全テストメソッドを修正する必要がある。** 影響は `plot_intraday` を呼ぶすべてのテストケース（現行で `ticker="7203.T"` を渡している箇所）。
- **`mpf.show()` はヘッドレス環境（CI）でブロックするかエラーになる可能性がある。** 現行も同じ問題があり、CI ではバックエンドを `Agg` に設定することで回避できる（既存の構成と同様）。
