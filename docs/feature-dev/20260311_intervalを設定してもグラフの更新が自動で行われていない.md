# intervalを設定してもグラフの更新が自動で行われていない

- Date: 2026-03-11
- Why needed: グラフを自動で更新するようにしたい。またその際の予測表示も自動で更新するようにしたい。なぜならユーザーは起動したまま置いておきたいから

## Scope

### In-scope
- interval 設定が有効なとき、指定間隔でグラフを自動再描画する
- グラフ再描画時に予測表示も合わせて自動更新する
- アプリ起動中は自動更新ループが継続して動作する

### Out-of-scope
- interval 設定 UI の変更・追加
- 自動更新の一時停止・再開機能
- 複数銘柄の同時自動更新

## Success criteria

- interval に設定した秒数ごとにグラフが自動で再描画される
- 再描画のたびに最新データを取得しチャートに反映される
- 予測ラインも再描画時に更新される
- アプリを終了しない限り自動更新ループが止まらない
- 既存の手動更新フローや他の表示機能が壊れない

## Risks/unknowns

- 現在の自動更新ループが実際に動いていない原因の特定が必要（タイマー未起動、コールバック未登録など）
- matplotlib の animation/timer API と既存描画コードの整合性
- データ取得が interval より遅い場合の競合・スキップ処理
- 予測計算がリアルタイム更新に対して十分高速かどうか
