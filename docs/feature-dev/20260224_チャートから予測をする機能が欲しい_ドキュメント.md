# チャートから予測をする機能が欲しい - ドキュメント

- **Date:** 2026-02-24

---

## Overview

`--predict` フラグを CLI に追加し、ローソク足チャート上にテクニカルパターンの自動検出結果と将来予測ラインをオーバーレイ表示する機能を追加する。
パターン検出はルールベース（numpy による極値検出）で行い、外部 ML ライブラリへの依存は追加しない。
`plot_intraday()`（静止画モード）と `plot_intraday_live()`（自動更新モード）の両方で動作する。

---

## Goals / Non-Goals

### Goals

- `--predict` フラグ（`store_true`）を `main.py` の CLI に追加する
- `src/predictor.py` を新規作成し、パターン検出・予測ライン生成・描画を担う
- 対象パターン: ダブルトップ / ダブルボトム / ヘッドアンドショルダー / 逆ヘッドアンドショルダー
- 予測ラインは将来 10 本（点線）をローソク足チャートの時間軸延長上にオーバーレイ描画
- パターン名を `axes[0]` 上に `ax.annotate()` で表示
- パターン未検出時は Row 3 meta テキストに「パターン未検出」を付加して表示
- `plot_intraday_live()` の `animate()` では `ax.clear()` 後に毎フレーム予測を再描画
- 既存の 79 件のテストを維持したまま新機能を追加する

### Non-Goals

- ML / ディープラーニングによる予測
- scipy への依存追加
- 複数パターンの同時描画（最強シグナル 1 件のみ）
- バックテスト・予測精度評価
- パラメータのユーザー設定

## Proposed Approach

### 1. 新規モジュール `src/predictor.py`

#### データクラス

```python
from dataclasses import dataclass
from typing import Optional
import numpy as np
import pandas as pd

@dataclass
class PatternResult:
    name:            str               # 例: "ダブルトップ"
    direction:       str               # "up" または "down"
    forecast_prices: list[float]       # 予測価格列（10本）
    forecast_index:  pd.DatetimeIndex  # 予測時刻列（将来タイムスタンプ）
```

#### 極値検出ヘルパー（scipy 不使用）

`order` 本の前後で最大・最小を持つ点を極値とする。

```python

    """Close 系列からチャートパターンを検出する。
```

検出順序（最初にマッチしたものを返す）:

| 優先度 | パターン | 条件 |
|---|---|---|
| 1 | ヘッドアンドショルダー | 極大3つ: 中央が最高、両肩の価格差 ≤ 3% |
| 2 | 逆ヘッドアンドショルダー | 極小3つ: 中央が最低、両肩の価格差 ≤ 3% |
| 3 | ダブルトップ | 極大2つ: 価格差 ≤ 2%、間に極小が存在 |
| 4 | ダブルボトム | 極小2つ: 価格差 ≤ 2%、間に極大が存在 |

#### 将来タイムスタンプ生成

```python
def _extrapolate_index(df: pd.DataFrame, n_steps: int = 10) -> pd.DatetimeIndex:
    last_ts = df.index[-1]
    freq = pd.tseries.frequencies.to_offset("1min")
    return pd.date_range(start=last_ts + freq, periods=n_steps, freq=freq, tz=last_ts.tz)
```

- 最後の Close から direction に応じて線形外挿（上: +0.5% × n / 下: -0.5% × n）
- `forecast_prices` は 10 本分の float リスト

```python

    """予測ラインとパターン名を ax に描画する。"""
```

- 予測ライン: `ax.plot(result.forecast_index, result.forecast_prices, ...)`
  - `linestyle="--"`, `alpha=0.7`, `color="#e67e22"`（橙）
- パターン名: `ax.annotate(result.name, ...)` をチャート右上付近に表示

### 2. `src/chart.py` の変更

#### `plot_intraday()` の変更点

| 箇所 | 変更前 | 変更後 |
|---|---|---|
| シグネチャ | `plot_intraday(df, info)` | `plot_intraday(df, info, predict: bool = False)` |
| axes 取得 | `fig, _ = mpf.plot(...)` | `fig, axes = mpf.plot(...)` |
| 予測オーバーレイ | なし | `predict=True` 時に `detect_pattern` → `draw_prediction` を呼ぶ |
| meta テキスト | `parts["meta"]` をそのまま表示 | `predict=True` かつ未検出時は `"  [パターン未検出]"` を付加 |

追加コード（`mpf.show()` の直前）:

```python
if predict:
    from src.predictor import detect_pattern, draw_prediction
    result = detect_pattern(plot_df)
    if result:
        draw_prediction(axes[0], result)
    else:
        _meta_text_obj.set_text(parts["meta"] + "  [パターン未検出]")
```

> `_meta_text_obj` は `fig.text(0.01, 0.885, ...)` の戻り値を保持したもの。

#### `plot_intraday_live()` の変更点

| 箇所 | 変更前 | 変更後 |
|---|---|---|
| シグネチャ | `plot_intraday_live(code, info, interval_sec)` | `predict: bool = False` を追加 |
| `animate()` 内 | `_meta_txt.set_text(...)` で終わる | `predict=True` 時に `detect_pattern` → `draw_prediction` を再呼び出し |

`animate()` 末尾の追加コード:

```python
if predict:
    from src.predictor import detect_pattern, draw_prediction
    result = detect_pattern(plot_df_new)
    if result:
        draw_prediction(ax1, result)
    else:
        _meta_txt.set_text(new_parts["meta"] + "  [パターン未検出]")
```
