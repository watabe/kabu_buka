# Yahoo!ファイナンスのようにもっと企業情報をわかりやすく表示させたい

- **Date:** 2026-02-24

## Why Needed

現状はただ上に表示しただけなので、フォントサイズや位置などを工夫して一見でわかるように表示させてください。

`fig.suptitle()` で1行テキストを上部に移動したが、情報が均一なサイズ・同一フォントで横並びになっているだけで視認性が低い。Yahoo!ファイナンスの銘柄ページのように、銘柄名・現在値・前日比などの重要度に応じてフォントサイズや色を変え、チャートを一目見ただけで必要な情報が把握できるレイアウトを実現したい。

---

## Scope

### In-scope
- チャート上部（`fig.suptitle()` に代わる実装）の情報表示レイアウト改善
  - 銘柄名・銘柄コードを大きく表示
  - 現在値を最も目立つサイズで表示
  - 前日比（差額・パーセント）をプラスは緑・マイナスは赤で色付け
  - 出来高・時価総額・取得日時は補足情報として小さく表示
- `plot_intraday()` と `plot_intraday_live()` の両方に適用
- `plot_intraday_live()` 自動更新時に情報テキストも更新される
- 色・フォントサイズは日本語フォント解決済みの環境で正しく表示される

### Out-of-scope
- CUI（コンソール）側の表示フォーマット変更
- チャート外部パネルやウィジェットの追加
- 銘柄ごとに配色テーマを変える設定機能
- 複数行レイアウト（マルチライン情報パネル）の複雑な配置

---

## Success Criteria

- 銘柄名と現在値が他の情報より大きいフォントサイズで表示される
- 前日比がプラスのとき緑（例: `#22ab94`）、マイナスのとき赤（例: `#f7525f`）で表示される
- 出来高・時価総額・取得日時は小さいフォントサイズで補足表示される
- `plot_intraday_live()` で自動更新のたびに表示内容・色が最新データで更新される
- 既存の 70 件のテストが引き続き PASS する
- 追加・変更したテストが PASS する

---

## Risks/Unknowns

- `fig.suptitle()` は単一 `Text` オブジェクトのため色分けや異なるフォントサイズを1行内に混在させることができない → `fig.text()` を複数配置するか `matplotlib.lines.Line2D` 等で代替する必要がある
- 複数の `fig.text()` を使う場合、`plot_intraday_live()` の `animate()` 内での更新が複数オブジェクトへの `set_text()` / `set_color()` 呼び出しに増える
- x 座標・y 座標のハードコードでレイアウトが組みにくい場合、figsize 変更時にずれる可能性がある
- テスト側で複数の `fig.text()` 呼び出しの引数を個別に検証する必要があり、テスト修正コストが増える可能性がある
