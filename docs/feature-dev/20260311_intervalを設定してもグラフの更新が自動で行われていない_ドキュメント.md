# intervalを設定してもグラフの更新が自動で行われていない - 機能ドキュメント

- **Date:** 2026-03-11

---

## Overview

調査結果（`20260311_..._調査結果.md`）で判明した 4 つの問題を修正し、`--interval` 設定時にグラフと予測表示が指定間隔で自動更新されるようにする。

変更は `src/chart.py` の `plot_intraday_live()` および内部の `animate()` ローカル関数に集中する。`main.py` / `tests/test_chart.py` は最小変更で対応する。

---

## Goals / Non-Goals

### Goals

- `--interval N` を指定したとき、N 秒ごとにローソク足チャートが実際に再描画される
- 更新のたびに `predict=True` の予測ラインも再計算・再描画される
- グラフ上部の現在値・前日比・出来高テキストも最新データに更新される
- アプリを終了しない限り自動更新ループが止まらない
- 既存の手動 1 回表示（`--chart` のみ）の挙動は変わらない

### Non-Goals

- `--interval` 設定 UI の変更
- ネットワーク遅延を考慮したスキップ・キューイング処理
- `fetch_stock_info()` の再呼び出しによる時価総額・前日終値の再取得

---

## Proposed Approach

### 問題1・2 の修正: `mpf.show()` → `plt.show(block=True)`

`mpf.show()` を `matplotlib.pyplot.show(block=True)` に置き換える。これにより：

- GUI バックエンドが interactive モードでも必ずブロッキング動作になる
- ブロッキング中は `ani` がローカルスコープに残り GC されない
- `FuncAnimation` タイマーがイベントループ内で確実に発火する

```python
# 変更前
mpf.show()

# 変更後
import matplotlib.pyplot as plt
# ...
plt.show(block=True)
```

### 問題3 の修正: `animate()` に `fig.canvas.draw_idle()` を追加

`ax.clear()` + `mpf.plot(ax=ax1, ...)` の後に `fig.canvas.draw_idle()` を呼び、バックエンドに再描画を明示的に要求する。

```python
def animate(_ival: int) -> None:
    df_new = fetch_intraday_data(code)
    if df_new.empty:
        return
    plot_df_new = df_new[["Open", "High", "Low", "Close", "Volume"]].copy()
    ax1.clear()
    ax2.clear()
    mpf.plot(plot_df_new, ax=ax1, volume=ax2, type="candle", style=style)
    # ... テキスト更新 ...
    # predict 処理 ...
    fig.canvas.draw_idle()   # ← 追加
```

### 問題4 の修正: `animate()` 内で `info` の現在値・出来高を最新データで更新

`fetch_intraday_data()` は OHLCV を返すため、最終行の Close・Volume 合計を `dataclasses.replace()` で `info` に反映した `info_live` を生成し、`_build_info_parts()` に渡す。前日終値・時価総額・会社名は変化しないため初期 `info` から引き継ぐ。

```python
from dataclasses import replace

def animate(_ival: int) -> None:
    df_new = fetch_intraday_data(code)
    if df_new.empty:
        return
    plot_df_new = df_new[["Open", "High", "Low", "Close", "Volume"]].copy()
    ax1.clear()
    ax2.clear()
    mpf.plot(plot_df_new, ax=ax1, volume=ax2, type="candle", style=style)

    info_live = replace(
        info,
        current_price=float(plot_df_new["Close"].iloc[-1]),
        volume=int(plot_df_new["Volume"].sum()),
    )
    new_parts = _build_info_parts(plot_df_new, info_live)
    _price_txt.set_text(new_parts["price"])
    _change_txt.set_text(new_parts["change"])
    _change_txt.set_color(new_parts["change_color"])
    _meta_txt.set_text(new_parts["meta"])

    if predict:
        from src.predictor import detect_pattern, draw_prediction
        result = detect_pattern(plot_df_new)
        if result:
            draw_prediction(ax1, result, n_existing=len(plot_df_new))
        else:
            ax1.text(
                0.98, 0.97, "パターン未検出",
                transform=ax1.transAxes, ha="right", va="top",
                fontsize=10, color="#888888",
                bbox=dict(boxstyle="round,pad=0.3", facecolor="white", alpha=0.7),
            )
    fig.canvas.draw_idle()
```

## Data Model Changes

`StockInfo` dataclass に変更なし。`animate()` 内でのみ `dataclasses.replace()` を使って一時オブジェクト `info_live` を生成する（副作用なし）。
