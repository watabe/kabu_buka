# 企業情報をわかりやすく表示させる - 機能ドキュメント

- **Date:** 2026-02-22

## Overview

`src/chart.py` の `plot_intraday()` と `plot_intraday_live()` において、現在チャート**下部**に小さく表示している会社情報テキスト（銘柄名・現在値・前日比・出来高・時価総額・取得日時）を、チャート**上部**（`fig.suptitle()`）へ移動する。

これにより、チャートウィンドウを開いた瞬間に「どの銘柄・現在値・更新日時」が視認でき、「チャートをみているだけで株価がわかる状態」を実現する。

変更は `src/chart.py` のみで完結する。`main.py`・`src/models.py`・`src/fetcher.py` への変更はない。

---

## Goals / Non-Goals

### Goals

- `plot_intraday()` で表示するチャートの**上部**に、銘柄名・現在値・前日比・出来高・時価総額・取得日時が表示される
- `plot_intraday_live()` で、自動更新のたびに**上部**の情報テキストが最新データで更新される
- 既存の下部 `fig.text()` は削除し、上部 `fig.suptitle()` に一本化する
- `_build_info_text()` は変更せず再利用する

### Non-Goals

- CUI（コンソール）側の表示フォーマット変更
- `mpf.plot(title=...)` で設定している ticker + 期間ラベルの変更
- チャート以外のウィンドウや別パネルの追加

---

## Proposed Approach

### 変更概要

#### `src/chart.py` — `plot_intraday()` の変更

**変更前（下部テキスト）:**

```python
fig, _ = mpf.plot(
    plot_df,
    ...
    returnfig=True,
)
fig.subplots_adjust(bottom=0.12)
fig.text(0.01, 0.02, _build_info_text(plot_df, info), fontsize=9, va="bottom")
mpf.show()
```

**変更後（上部タイトル）:**

```python
fig, _ = mpf.plot(
    plot_df,
    ...
    returnfig=True,
)
fig.subplots_adjust(top=0.88)
fig.suptitle(_build_info_text(plot_df, info), fontsize=11, y=0.98)
mpf.show()
```

変更点:
- `fig.subplots_adjust(bottom=0.12)` → `fig.subplots_adjust(top=0.88)` （上部マージン確保）
- `fig.text(0.01, 0.02, ..., fontsize=9, va="bottom")` → `fig.suptitle(..., fontsize=11, y=0.98)` （上部タイトルに移動・フォントサイズ拡大）

#### `src/chart.py` — `plot_intraday_live()` の変更

**変更前（下部テキスト）:**

```python
fig.subplots_adjust(bottom=0.12)
info_txt = fig.text(0.01, 0.02, _build_info_text(plot_df, info),
                    fontsize=9, va="bottom")

def animate(_ival: int) -> None:
    df_new = fetch_intraday_data(code)
    if df_new.empty:
        return
    plot_df_new = df_new[["Open", "High", "Low", "Close", "Volume"]].copy()
    ax1.clear()
    ax2.clear()
    mpf.plot(plot_df_new, ax=ax1, volume=ax2, type="candle", style=style)
    info_txt.set_text(_build_info_text(plot_df_new, info))
```

**変更後（上部タイトル）:**

```python
fig.subplots_adjust(top=0.88)
suptitle_txt = fig.suptitle(_build_info_text(plot_df, info), fontsize=11, y=0.98)

def animate(_ival: int) -> None:
    df_new = fetch_intraday_data(code)
    if df_new.empty:
        return
    plot_df_new = df_new[["Open", "High", "Low", "Close", "Volume"]].copy()
    ax1.clear()
    ax2.clear()
    mpf.plot(plot_df_new, ax=ax1, volume=ax2, type="candle", style=style)
    suptitle_txt.set_text(_build_info_text(plot_df_new, info))
```

変更点:
- `fig.subplots_adjust(bottom=0.12)` → `fig.subplots_adjust(top=0.88)`
- `info_txt = fig.text(...)` → `suptitle_txt = fig.suptitle(...)`
- `info_txt.set_text(...)` → `suptitle_txt.set_text(...)`

> **設計根拠:** `fig.suptitle()` はフィギュアレベルのアーティストであり、`ax.clear()` の影響を受けない。`animate()` 内で `ax1.clear()` / `ax2.clear()` を呼んでも `suptitle_txt` は消えず、`set_text()` でインプレース更新できる。
