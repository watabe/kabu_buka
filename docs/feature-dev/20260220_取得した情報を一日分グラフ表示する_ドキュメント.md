# 取得した情報を一日分グラフ表示する - 機能ドキュメント

- **Date:** 2026-02-20

## Overview

既存の CLI 銘柄情報ツール（`main.py`）に `--chart` フラグを追加し、`mplfinance` を使って指定銘柄の当日1分足データをローソク足チャート（出来高サブプロット付き）でネイティブGUIウィンドウに表示する。

既存のテキスト表示（`print_stock_info`）はそのまま動作し、`--chart` を付けたときだけグラフが追加表示される。

---

## Goals / Non-Goals

### Goals
- `python main.py 7203 --chart` で当日1分足のローソク足チャートを表示する
- 時刻軸（JST）・株価軸・出来高サブプロットをラベル付きで描画する
- 上昇ローソク（陽線）・下降ローソク（陰線）が色で判別できる（`yahoo` スタイル）
- 既存のテキスト出力はそのまま維持し、グラフは追加表示として機能する
- 取引時間外・データなし時は適切なエラーメッセージを出して終了する

### Non-Goals
- 折れ線グラフ・OHLC バーなど複数チャート種別の切り替え（v1 はローソク足固定）
- インターバル変更オプション（v1 は 1m 固定）
- 複数日・長期チャートの表示
- グラフのズーム・パン等のインタラクティブ操作
- グラフ画像のファイル保存
- リアルタイム自動更新

## Proposed Approach

### 技術選定

| 項目 | 選択 | 理由 |
|---|---|---|
| チャートライブラリ | `mplfinance>=0.12.10` | ローソク足を1行で描画。yfinance DataFrame と直接互換。ブラウザ不要のネイティブウィンドウ |
| intraday データ取得 | `yf.Ticker.history(period="1d", interval="1m")` | yfinance で直近7日以内の1分足を取得可能 |
| GUI バックエンド | matplotlib デフォルト（`MacOSX` / `TkAgg`） | `MPLBACKEND=TkAgg` で環境依存問題を回避可能 |

### 処理フロー

```
python main.py 7203 --chart
    ↓
main.py: argparse で code + --chart フラグを受け取る
    ↓
fetch_stock_info("7203") → StockInfo（テキスト表示）
print_stock_info(info) → CLIにテキスト出力（既存動作を維持）
    ↓（--chart フラグがある場合のみ）
fetch_intraday_data("7203") → pandas DataFrame（1分足 OHLCV）
    ↓
DataFrame が空？ → "当日の取引データがありません: 7203" を出力、exit 1
    ↓（データあり）
plot_intraday(df, ticker="7203.T") → mplfinance でウィンドウ表示
```

### CLI インターフェース

```bash
# テキストのみ（既存動作）
python main.py 7203

# テキスト + グラフ
python main.py 7203 --chart

# バックエンドを明示する場合（環境依存問題の回避）
MPLBACKEND=TkAgg python main.py 7203 --chart
```

### エラーハンドリング方針

| ケース | 挙動 |
|---|---|
| DataFrame が空（取引時間外・休日） | "当日の取引データがありません: {code}" を stderr に出力、exit 1 |
| ネットワークエラー | 既存の FetchError 処理と同様、"データ取得に失敗しました。..." を出力、exit 1 |
| GUI バックエンドエラー | 未キャッチ（環境設定の問題として `MPLBACKEND=TkAgg` を案内するメッセージを出力） |
