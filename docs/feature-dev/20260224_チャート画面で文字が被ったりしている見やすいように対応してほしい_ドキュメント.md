# チャート画面で文字が被ったりしている見やすいように対応してほしい - 機能ドキュメント

- **Date:** 2026-02-24

## Overview

調査により、文字衝突の根本原因は `mpf.plot(title=...)` が内部で `fig.suptitle(y=0.98)` を描画するのに対し、我々の `fig.text(y=0.975)` が直下に配置されていたためと判明した。

本フィーチャーでは以下の2点を行う:

1. **`mpf.plot()` の `title=` を空文字 `""` に変更**して mplfinance による suptitle 描画を抑制する
2. **3行レイアウトに再設計**して情報の優先度・可読性を改善し、あらゆる銘柄名の長さでも衝突が起きない構造にする

変更は `src/chart.py` のみで完結する。

---

## Goals / Non-Goals

### Goals

- チャートウィンドウを開いたとき、すべての情報テキストが重ならない
- `mpf.plot(title=...)` に起因する suptitle と `fig.text` の衝突を完全に解消する
- 人間工学的な情報優先度に従い、第1視線（銘柄名・期間）→ 第2視線（現在値・前日比）→ 第3視線（補足）の3行構造にする
- `plot_intraday()` と `plot_intraday_live()` の両方に適用する
- `plot_intraday_live()` 自動更新時に Row 2・Row 3 が最新データで更新される

### Non-Goals

- `_build_info_parts()` の返す辞書の変更（`name`/`price`/`change`/`change_color`/`meta` フィールドは保持）
- figsize の変更
- CUI 側の表示変更

---

## Proposed Approach

### レイアウト設計

```
[figure top = 1.0]
  y=0.960  Row 1:  {info.name} ({info.ticker})  1日 (1分足)      fontsize=13, bold, color=#333333
  y=0.922  Row 2:  ¥{price}                      fontsize=16, bold, color=#333333
           Row 2:  {sign}{diff} ({sign}{pct}%)   fontsize=12, color=#22ab94 or #f7525f  (x=0.20)
  y=0.885  Row 3:  出来高: xxx  時価総額: xxx  取得: HH:MM        fontsize=9,  color=#666666
[subplots_adjust top = 0.85]
[chart area]
```

Row 2 は `price` (x=0.01) と `change` (x=0.20) を同一 y 行に横並びにする。現在値の最長表示 "¥999,999"（7文字 × 16pt ≈ 0.11 figure 幅）に対し x=0.20 から change を開始すれば十分なマージンが確保できる。

### `plot_intraday()` の変更

**変更前:**
```python
fig, _ = mpf.plot(
    plot_df, ...,
    title=f"{info.ticker}  1日 (1分足)",
    ...
)
fig.subplots_adjust(top=0.82)
parts = _build_info_parts(plot_df, info)
fig.text(0.01, 0.975, parts["name"],   fontsize=13, fontweight="bold", va="top", color="#333333")
fig.text(0.42, 0.975, parts["price"],  fontsize=16, fontweight="bold", va="top", color="#333333")
fig.text(0.58, 0.975, parts["change"], fontsize=12, va="top", color=parts["change_color"])
fig.text(0.01, 0.930, parts["meta"],   fontsize=9,  va="top", color="#666666")
```

**変更後:**
```python
fig, _ = mpf.plot(
    plot_df, ...,
    title="",          # mplfinance の suptitle を抑制
    ...
)
fig.subplots_adjust(top=0.85)
parts = _build_info_parts(plot_df, info)
fig.text(0.01, 0.960, parts["name"] + "  1日 (1分足)",  fontsize=13, fontweight="bold", va="top", color="#333333")
fig.text(0.01, 0.922, parts["price"],                   fontsize=16, fontweight="bold", va="top", color="#333333")
fig.text(0.20, 0.922, parts["change"],                  fontsize=12, va="top", color=parts["change_color"])
fig.text(0.01, 0.885, parts["meta"],                    fontsize=9,  va="top", color="#666666")
```

### `plot_intraday_live()` の変更

**変更前:**
```python
fig, axes = mpf.plot(
    plot_df, ...,
    title=f"{info.ticker}  1日 (1分足)  自動更新",
    ...
)
fig.subplots_adjust(top=0.82)
parts = _build_info_parts(plot_df, info)
fig.text(0.01, 0.975, parts["name"],   fontsize=13, fontweight="bold", va="top", color="#333333")
_price_txt  = fig.text(0.42, 0.975, parts["price"],  fontsize=16, fontweight="bold", va="top", color="#333333")
_change_txt = fig.text(0.58, 0.975, parts["change"], fontsize=12, va="top", color=parts["change_color"])
_meta_txt   = fig.text(0.01, 0.930, parts["meta"],   fontsize=9,  va="top", color="#666666")

def animate(_ival: int) -> None:
    ...
    new_parts = _build_info_parts(plot_df_new, info)
    _price_txt.set_text(new_parts["price"])
    _change_txt.set_text(new_parts["change"])
    _change_txt.set_color(new_parts["change_color"])
    _meta_txt.set_text(new_parts["meta"])
```

**変更後:**
```python
fig, axes = mpf.plot(
    plot_df, ...,
    title="",          # mplfinance の suptitle を抑制
    ...
)
fig.subplots_adjust(top=0.85)
parts = _build_info_parts(plot_df, info)
fig.text(0.01, 0.960, parts["name"] + "  1日 (1分足)  自動更新",

_price_txt  = fig.text(0.01, 0.922, parts["price"],  fontsize=16, fontweight="bold", va="top", color="#333333")
_change_txt = fig.text(0.20, 0.922, parts["change"], fontsize=12, va="top", color=parts["change_color"])
_meta_txt   = fig.text(0.01, 0.885, parts["meta"],   fontsize=9,  va="top", color="#666666")

def animate(_ival: int) -> None:
    ...
    new_parts = _build_info_parts(plot_df_new, info)
    _price_txt.set_text(new_parts["price"])
    _change_txt.set_text(new_parts["change"])
    _change_txt.set_color(new_parts["change_color"])

```

> `animate()` 内の更新対象は変更なし（`_price_txt` / `_change_txt` / `_meta_txt` の3オブジェクト）。Row 1 名前テキストは変化しないため更新不要。
