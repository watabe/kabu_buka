# Yahoo!ファイナンスの情報取得

- **Date:** 2026-02-20

## Why Needed

指定した銘柄コードで情報を取得し画面上に表示する

## Scope

### In-scope
- 銘柄コードを入力として受け取る機能
- Yahoo!ファイナンスから銘柄情報を取得する処理
- 取得した情報を画面上に表示する機能

### Out-of-scope
- リアルタイムのティッカー表示（自動更新）
- ポートフォリオ管理機能
- 銘柄情報の永続化・保存
- 複数銘柄の一括取得

## Success Criteria

- 銘柄コードを入力すると、対応するYahoo!ファイナンスの情報が正常に取得できる
- 取得した銘柄情報（銘柄名、株価、前日比など）が画面に表示される
- 存在しない銘柄コードを入力した場合、適切なエラーメッセージが表示される
- 取得処理中はローディング状態が適切に表示される

## Risks / Unknowns

- Yahoo!ファイナンスのスクレイピング規約・利用規約の確認が必要
- Yahoo!ファイナンスのHTML構造が変更された場合、取得処理が壊れる可能性がある
- ネットワークエラーや取得タイムアウト時の挙動の設計が必要
- 取得レート制限（短時間に大量リクエストを行った場合のブロック）への対応
