# チャート表示の自動更新 - 機能ドキュメント

- **Date:** 2026-02-21

## Overview

`src/chart.py` に新関数 `plot_intraday_live()` を追加し、`matplotlib.animation.FuncAnimation` と mplfinance の External Axes Mode を組み合わせて、指定インターバル（デフォルト60秒）ごとに yfinance からデータを再取得してローソク足チャートを自動更新する。

既存の `plot_intraday()`（1回表示）は変更せず温存する。`main.py` に `--interval` オプションを追加し、指定時は `plot_intraday_live()` を呼ぶ。変更は `src/chart.py`・`main.py` の2ファイルで完結する。

---

## Goals / Non-Goals

### Goals
- `--interval N` で更新間隔（秒）を指定でき、チャートが N 秒ごとに自動更新される
- デフォルト（`--interval` 未指定）は従来どおり1回表示で終了する
- 更新のたびに会社情報テキスト（現在値・前日比・取得日時など）も最新データで更新される
- 取引時間外など再取得データが空でもクラッシュせず前回データを表示し続ける
- ウィンドウを閉じるか Ctrl+C するとループが終了しプログラムが正常終了する

### Non-Goals
- `--interval` なし `--chart` の1回表示モードの動作変更
- WebSocket・ストリーミング API によるミリ秒単位のリアルタイム更新
- 複数銘柄の同時表示・同時更新
- チャートウィンドウの常駐化・システムトレイ格納

**新関数 `plot_intraday_live()`**

```python
def plot_intraday_live(code: str, info: StockInfo, interval_sec: int = 60) -> None:
    """指定インターバルごとにチャートを自動更新して表示する。

    ウィンドウを閉じるか Ctrl+C するまでループし続ける。
    取得データが空のフレームはスキップして前回の描画を維持する。

    Args:
        code:         東証銘柄コード (例: "7203")
        info:         初期表示用の銘柄情報（StockInfo）
        interval_sec: 更新インターバル（秒）。デフォルト 60 秒

    Raises:
        ValueError: 初回データが空の場合
    """
    df_init = fetch_intraday_data(code)
    if df_init.empty:
        raise ValueError(f"表示するデータがありません: {info.ticker}")

    font = _resolve_japanese_font()
    rc: dict = {"axes.unicode_minus": False}
    if font:
        rc["font.family"] = font
    style = mpf.make_mpf_style(base_mpf_style="yahoo", rc=rc)

    plot_df = df_init[["Open", "High", "Low", "Close", "Volume"]].copy()
    fig, axes = mpf.plot(
        plot_df,
        type="candle",
        style=style,
        title=f"{info.ticker}  1日 (1分足)  自動更新",
        ylabel="株価 (円)",
        ylabel_lower="出来高",
        volume=True,
        figsize=(14, 7),
        show_nontrading=False,
        returnfig=True,
    )
    ax1 = axes[0]
    ax2 = axes[2]

    fig.subplots_adjust(bottom=0.12)
    info_txt = fig.text(0.01, 0.02, _build_info_text(df_init, info),
                        fontsize=9, va="bottom")

    def animate(_ival: int) -> None:
        df_new = fetch_intraday_data(code)
        if df_new.empty:
            return
        plot_df_new = df_new[["Open", "High", "Low", "Close", "Volume"]].copy()
        ax1.clear()
        ax2.clear()
        mpf.plot(plot_df_new, ax=ax1, volume=ax2,
                 type="candle", style=style)
        info_txt.set_text(_build_info_text(df_new, info))

    ani = animation.FuncAnimation(  # noqa: F841  (ani はスコープ保持が必要)
        fig, animate,
        interval=interval_sec * 1000,
        cache_frame_data=False,
    )
    mpf.show()
```

**プライベートヘルパー `_build_info_text()`**

```python
def _build_info_text(df: pd.DataFrame, info: StockInfo) -> str:
    """チャート下部に表示する情報テキストを組み立てる。"""
    diff = info.current_price - info.previous_close
    pct  = (diff / info.previous_close * 100) if info.previous_close else 0.0
    sign = "+" if diff >= 0 else ""
    cap  = _format_market_cap(info.market_cap)
    ts   = df.index[-1].strftime("%Y-%m-%d %H:%M")
    return (
        f"{info.name} ({info.ticker})  "
        f"現在値: ¥{info.current_price:,.0f}  "
        f"前日比: {sign}{diff:,.0f} ({sign}{pct:.2f}%)  "
        f"出来高: {info.volume:,}  "
        f"時価総額: {_format_market_cap(info.market_cap)}  "
        f"取得: {ts}"
    )
```

> `_build_info_text()` は `plot_intraday()` の既存インライン処理をリファクタリングして共通化する。`plot_intraday()` 内のインライン文字列構築も `_build_info_text()` 呼び出しに置き換える。

#### `main.py` — 変更

```python
# 追加引数
parser.add_argument(
    "--interval",
    type=int,
    default=None,
    metavar="SEC",
    help="チャートを N 秒ごとに自動更新する（例: --interval 60）",
)

# 呼び出し分岐
if args.interval is not None:
    plot_intraday_live(args.code, info=info, interval_sec=args.interval)
else:
    plot_intraday(df, info=info)
```

> `--interval` を指定した場合は `fetch_intraday_data` を `plot_intraday_live` 内部で呼ぶため、`main.py` 側の `fetch_intraday_data` 呼び出しは不要になる。
