# 予測ラインをパターン理論値に直し、信頼度を表示する - 機能ドキュメント

- **Date:** 2026-03-12

---

## Overview

`src/predictor.py` の予測ライン生成を、全パターン共通の固定 0.5%/本 から**各パターンの理論的な目標価格**に基づくものに変更する。あわせて `PatternResult` に `confidence` フィールドを追加し、`draw_prediction()` のパターン名表示に信頼度（パーセント）を付加する。

変更は `src/predictor.py` に集中し、`src/chart.py` / `main.py` は変更不要。

---

## Goals / Non-Goals

### Goals

- `PatternResult` dataclass に `confidence: float`（0.0〜1.0）フィールドを追加する
- `_make_forecast()` を廃止し、`_make_target_forecast(last_close, target_price, n)` に置き換える
  - `last_close` から `target_price` まで `n` ステップで線形補間する
- 各パターン検出ブロックでパターン固有の目標価格と信頼度を計算する
- `draw_prediction()` のパターン名テキストを `"パターン名 (XX%)"` 形式にする
- 既存テストを最小限の変更で維持しつつ、新テストで理論値・信頼度を検証する

### Non-Goals

- `src/chart.py` / `main.py` の変更
- 信頼度の ML・統計モデルへの置き換え
- ネックライン割れの確認（シグナル確定の遅延処理）
- 出来高フィルタ
- 可変本数予測（`forecast_prices` は固定 10 本のまま）

## Proposed Approach

### 1. `PatternResult` に `confidence` フィールドを追加

```python
@dataclass
class PatternResult:
    name:            str
    direction:       str
    forecast_prices: list
    forecast_index:  pd.DatetimeIndex

```

必須フィールドとする（`dataclasses.field(default=...)` は使わない）。既存の `PatternResult(...)` コンストラクト呼び出し 13 箇所および `_make_pattern_result()` テストヘルパーをすべて更新する。

---

### 2. `_make_forecast()` を廃止 → `_make_target_forecast()` に置き換え

```python
def _make_target_forecast(last_close: float, target_price: float, n: int = 10) -> list:
    """last_close から target_price まで n ステップで線形補間した予測価格列を返す。"""
    return list(np.linspace(last_close, target_price, n + 1)[1:])
```

`np.linspace(start, stop, n+1)[1:]` により、スタート点を含まない n 点の等間隔リストを生成する。`target_price` の方向（up/down）は目標価格の大小で自動的に決まるため、`direction` を引数に取らない。

### 3. 各パターンの目標価格と信頼度の計算

#### ヘッドアンドショルダー

```python
# ネックライン: 左肩〜頭 および 頭〜右肩 の間にある極小の平均
neck_candidates = minima[(minima > l_idx) & (minima < r_idx)]
neckline = float(close[neck_candidates].mean()) if len(neck_candidates) > 0 else float(min(close[l_idx], close[r_idx]))
target   = neckline - (hv - neckline)
confidence = max(0.0, min(1.0, 1.0 - shoulder_diff / 0.03))
```

#### 逆ヘッドアンドショルダー

```python
neck_candidates = maxima[(maxima > l_idx) & (maxima < r_idx)]
neckline = float(close[neck_candidates].mean()) if len(neck_candidates) > 0 else float(max(close[l_idx], close[r_idx]))
target   = neckline + (neckline - hv)
confidence = max(0.0, min(1.0, 1.0 - shoulder_diff / 0.03))
```

#### ダブルトップ

```python
neckline   = float(close[between_minima].min())
peak_avg   = (v1 + v2) / 2.0
target     = neckline - (peak_avg - neckline)
confidence = max(0.0, min(1.0, 1.0 - peak_diff / 0.02))
```

#### ダブルボトム

```python
neckline   = float(close[between_maxima].max())
trough_avg = (v1 + v2) / 2.0
target     = neckline + (neckline - trough_avg)
confidence = max(0.0, min(1.0, 1.0 - trough_diff / 0.02))
```

#### トリプルトップ

```python
# ネックライン: b12 および b23 の谷の最安値
neckline   = min(float(close[b12].min()), float(close[b23].min()))
peak_avg   = (v1 + v2 + v3) / 3.0
target     = neckline - (peak_avg - neckline)
confidence = max(0.0, min(1.0, 1.0 - max(diff12, diff23) / 0.03))
```

#### トリプルボトム

```python
neckline   = max(float(close[b12].max()), float(close[b23].max()))
trough_avg = (v1 + v2 + v3) / 3.0
target     = neckline + (neckline - trough_avg)
confidence = max(0.0, min(1.0, 1.0 - max(diff12, diff23) / 0.03))
```

#### 上昇三角形 / 下降三角形 / 対称三角形 / ライジングウェッジ / フォーリングウェッジ

```python
# 三角形最大幅 = 最初の極大と最初の極小の価格差（最大振れ幅）
max_width  = abs(float(close[maxima[-3]]) - float(close[minima[-3]]))
# 上昇方向なら上、下降方向なら下
target     = last_close + max_width  # direction == "up"
target     = last_close - max_width  # direction == "down"

# 信頼度（三角形・ウェッジ共通）: 水平側傾きの「水平らしさ」または収束率
# 上昇三角形: 水平側は slope_high, 信頼度 = 1 - abs(slope_high)/flat_thresh にクランプ
# ウェッジ: 収束率 = min/max of abs slopes の比率
confidence = max(0.0, min(1.0, ...))  # パターンごとに後述
```

各パターンの信頼度式：

| パターン | 信頼度式 |
|---|---|
| 上昇三角形 | `1.0 - abs(slope_high) / flat_thresh` |
| 下降三角形 | `1.0 - abs(slope_low)  / flat_thresh` |
| 対称三角形 | `min(abs(slope_high), abs(slope_low)) / max(abs(slope_high), abs(slope_low))` ただし両方 > flat_thresh の場合 |
| ライジングウェッジ | `1.0 - slope_high / slope_low`（slope_low > slope_high のとき 0〜1 になる） |
| フォーリングウェッジ | `1.0 - slope_low / slope_high`（slope_high < slope_low < 0 のとき 0〜1 になる） |

#### 強気フラッグ / 弱気フラッグ

```python
pole_height = abs(float(pole[-1]) - float(pole[0]))
target      = last_close + pole_height   # 強気
target      = last_close - pole_height   # 弱気
confidence  = max(0.0, min(1.0, 1.0 - rest_range / (pole_range * 0.5)))
```

#### カップアンドハンドル

```python
cup_depth_abs = left_mean - mid_mean
target        = float(cup_right.max()) + cup_depth_abs
confidence    = max(0.0, min(1.0, 1.0 - handle_dip / 0.05))
```
