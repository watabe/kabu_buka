# 取得した情報を一日分グラフ表示する

- **Date:** 2026-02-20

## Why Needed

取得情報がCUIだけだと認知しにくいためGUIのグラフ情報として表示させる

## Scope

### In-scope
- 指定銘柄の当日分（1日分）の株価データをグラフとして表示する
- 時系列の株価推移（折れ線グラフまたはローソク足チャート）を可視化する
- 既存のCLI取得機能と連携して動作する
- GUIウィンドウまたはHTMLなどで視覚的に表示する

### Out-of-scope
- 複数日にわたる長期チャートの表示
- 複数銘柄の同時比較表示
- インタラクティブなズームや期間変更操作
- リアルタイム自動更新（グラフの動的更新）
- グラフ画像の保存・エクスポート機能

## Success Criteria

- 銘柄コードを入力すると、当日分の株価推移グラフがGUI上に表示される
- グラフに時刻軸と株価軸が正しくラベル表示される
- 株価の上昇・下降が視覚的に判別できる配色・スタイルで描画される
- 既存のCLIテキスト表示との併用が可能（グラフは追加表示として機能する）
- データ取得失敗時はグラフを表示せず適切なエラーメッセージを出す

## Risks / Unknowns

- 1日分の分足データ（intraday data）が yfinance の Free プランで取得可能か要確認
- 取引時間外・休日に実行した場合のデータ有無の扱いが未定
- GUIライブラリの選定（matplotlib / plotly / tkinter 等）が未決定
- ヘッドレス環境（サーバー・CI）でGUI表示できない場合の対応方針が未定
- 市場が閉まっている時間帯は当日データが存在せず、前日データにフォールバックするか否かが未定
