# チャート常にCUIで取得した会社情報も全て表示する - 機能ドキュメント

- **Date:** 2026-02-21

## Overview

`src/chart.py` の `plot_intraday()` に `StockInfo` オブジェクトを渡せるようシグネチャを変更し、ローソク足チャートウィンドウ内の下部に銘柄名・ティッカー・株価・前日比・出来高・時価総額・取得日時を1行テキストとして表示する。

`mplfinance` の `returnfig=True` オプションで `(fig, axes)` を取得し、`fig.subplots_adjust(bottom=0.12)` で余白を確保した後に `fig.text()` で情報を描画する。最後に `mpf.show()` でウィンドウを表示する。変更は `src/chart.py`・`main.py` の2ファイルで完結し、他のモジュールへの影響はない。

---

## Goals / Non-Goals

### Goals
- チャートウィンドウ内に銘柄名・ティッカーを表示し、「何の銘柄のグラフか」を一目でわかるようにする
- 取得日時（最終バーのタイムスタンプ）をチャートに明示し、「何日何時現在の情報か」を示す
- CUI テキスト出力と同等の主要指標（現在値・前日比・出来高・時価総額）をチャート内に表示する
- 日本語フォントが設定済みの環境でこれらのテキストが正しく表示される

### Non-Goals
- CUI テキスト出力フォーマットの変更
- リアルタイム更新・自動リフレッシュ
- チャートのレイアウト・配色の全面見直し
- `--chart` なしの通常起動時の動作変更
- 複数銘柄の同時表示

**`plot_intraday()` 実装変更**

```python
def plot_intraday(df: pd.DataFrame, info: StockInfo) -> None:
    if df.empty:
        raise ValueError(f"表示するデータがありません: {info.ticker}")

    plot_df = df[["Open", "High", "Low", "Close", "Volume"]].copy()

    font = _resolve_japanese_font()
    rc: dict = {"axes.unicode_minus": False}
    if font:
        rc["font.family"] = font

    style = mpf.make_mpf_style(base_mpf_style="yahoo", rc=rc)

    fig, _ = mpf.plot(
        plot_df,
        type="candle",
        style=style,
        title=f"{info.ticker}  1日 (1分足)",
        ylabel="株価 (円)",
        ylabel_lower="出来高",
        volume=True,
        figsize=(14, 7),
        show_nontrading=False,
        returnfig=True,
    )

    # 情報テキストの組み立て
    diff = info.current_price - info.previous_close
    pct  = (diff / info.previous_close * 100) if info.previous_close else 0.0
    sign = "+" if diff >= 0 else ""
    cap  = _format_market_cap(info.market_cap)
    ts   = df.index[-1].strftime("%Y-%m-%d %H:%M")
    info_text = (
        f"{info.name} ({info.ticker})  "
        f"現在値: ¥{info.current_price:,.0f}  "
        f"前日比: {sign}{diff:,.0f} ({sign}{pct:.2f}%)  "
        f"出来高: {info.volume:,}  "
        f"時価総額: {cap}  "
        f"取得: {ts}"
    )

    fig.subplots_adjust(bottom=0.12)
    fig.text(0.01, 0.02, info_text, fontsize=9, va="bottom")

    mpf.show()
```

`_format_market_cap()` は `src/display.py` に定義済みの関数と同一ロジックを使う。`display.py` からのインポートは循環依存の懸念がないため許容するか、同一ロジックを `chart.py` 内のプライベート関数として持つかは実装時に判断する（後者の方が依存が少なくシンプル）。

## API or Interface Changes

| 変更対象 | 変更前 | 変更後 |
|---|---|---|
| `plot_intraday(df, ticker)` | `ticker: str` | `info: StockInfo` |
| `mpf.plot(...)` 内部 | `returnfig` なし（暗黙 show） | `returnfig=True` + `mpf.show()` 明示 |

- **外部 API への影響なし**（`plot_intraday` はモジュール内部からのみ呼ばれ、公開 API ではない）
- `main.py` の呼び出し箇所を1行変更するだけ
