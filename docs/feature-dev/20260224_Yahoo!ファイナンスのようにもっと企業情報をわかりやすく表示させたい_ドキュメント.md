# Yahoo!ファイナンスのようにもっと企業情報をわかりやすく表示させたい - 機能ドキュメント

- **Date:** 2026-02-24

## Overview

`src/chart.py` において、現行の `fig.suptitle()` による単色・単一フォントサイズの1行テキスト表示を廃止し、複数の `fig.text()` オブジェクトを使ったリッチな2行レイアウトに置き換える。

- **Row 1（y=0.975）**: 銘柄名＋コード（太字大）・現在値（太字最大）・前日比（色付き中）
- **Row 2（y=0.930）**: 出来高・時価総額・取得日時（小・グレー）

前日比がプラスのときは緑（`#22ab94`）、マイナスのときは赤（`#f7525f`）で表示する。`plot_intraday_live()` の自動更新時は、各 `Text` オブジェクトの `set_text()` / `set_color()` でインプレース更新する。

変更は `src/chart.py` のみで完結する。

---

## Goals / Non-Goals

### Goals

- Row 1 に銘柄名・現在値・前日比を表示し、銘柄名は 13pt 太字、現在値は 16pt 太字、前日比は 12pt 色付きで表示される
- Row 2 に出来高・時価総額・取得日時を 9pt グレーで表示される
- 前日比プラス → 緑（`#22ab94`）、マイナス → 赤（`#f7525f`）で色付けされる
- `plot_intraday_live()` の自動更新のたびに現在値・前日比・取得日時・色が最新データで更新される
- 既存の `_build_info_text()` 関数・`TestBuildInfoText` テストは変更しない

### Non-Goals

- `mpf.plot(title=...)` で設定している ticker + 期間ラベルの変更
- figsize 変更への動的対応（x 座標ハードコード）
- HTML/CSS ライクな任意スタイリング
- CUI 側の表示変更

## Proposed Approach

### 新規ヘルパー `_build_info_parts()`

既存の `_build_info_text()` は保持した上で、新関数 `_build_info_parts()` を追加する。

```python
def _build_info_parts(df: pd.DataFrame, info: StockInfo) -> dict:
    """チャート上部の各テキストパーツを辞書で返す。

    Returns:
        {
            "name":         str,  # "{info.name} ({info.ticker})"
            "price":        str,  # "¥{current_price:,.0f}"
            "change":       str,  # "{sign}{diff:,.0f} ({sign}{pct:.2f}%)"
            "change_color": str,  # "#22ab94"（正）または "#f7525f"（負）
            "meta":         str,  # "出来高: xxx  時価総額: xxx  取得: YYYY-MM-DD HH:MM"
        }
    """
    diff = info.current_price - info.previous_close
    pct  = (diff / info.previous_close * 100) if info.previous_close else 0.0
    sign = "+" if diff >= 0 else ""
    ts   = df.index[-1].strftime("%Y-%m-%d %H:%M")
    return {
        "name":         f"{info.name} ({info.ticker})",
        "price":        f"¥{info.current_price:,.0f}",
        "change":       f"{sign}{diff:,.0f} ({sign}{pct:.2f}%)",
        "change_color": "#22ab94" if diff >= 0 else "#f7525f",
        "meta": (
            f"出来高: {info.volume:,}  "
            f"時価総額: {_format_market_cap(info.market_cap)}  "
            f"取得: {ts}"
        ),
    }
```

### `plot_intraday()` の変更

**変更前:**
```python
fig.subplots_adjust(top=0.88)
fig.suptitle(_build_info_text(plot_df, info), fontsize=11, y=0.98)
```

**変更後:**
```python
fig.subplots_adjust(top=0.82)
parts = _build_info_parts(plot_df, info)
fig.text(0.01, 0.975, parts["name"],   fontsize=13, fontweight="bold", va="top", color="#333333")
fig.text(0.42, 0.975, parts["price"],  fontsize=16, fontweight="bold", va="top", color="#333333")
fig.text(0.58, 0.975, parts["change"], fontsize=12, va="top", color=parts["change_color"])
fig.text(0.01, 0.930, parts["meta"],   fontsize=9,  va="top", color="#666666")
```

### `plot_intraday_live()` の変更

**変更前:**
```python
fig.subplots_adjust(top=0.88)
suptitle_txt = fig.suptitle(_build_info_text(plot_df, info), fontsize=11, y=0.98)

def animate(_ival: int) -> None:
    ...
    suptitle_txt.set_text(_build_info_text(plot_df_new, info))
```

**変更後:**
```python
fig.subplots_adjust(top=0.82)
parts = _build_info_parts(plot_df, info)
fig.text(0.01, 0.975, parts["name"],   fontsize=13, fontweight="bold", va="top", color="#333333")
_price_txt  = fig.text(0.42, 0.975, parts["price"],  fontsize=16, fontweight="bold", va="top", color="#333333")
_change_txt = fig.text(0.58, 0.975, parts["change"], fontsize=12, va="top", color=parts["change_color"])
_meta_txt   = fig.text(0.01, 0.930, parts["meta"],   fontsize=9,  va="top", color="#666666")

def animate(_ival: int) -> None:
    df_new = fetch_intraday_data(code)
    if df_new.empty:
        return
    plot_df_new = df_new[["Open", "High", "Low", "Close", "Volume"]].copy()
    ax1.clear()
    ax2.clear()
    mpf.plot(plot_df_new, ax=ax1, volume=ax2, type="candle", style=style)
    new_parts = _build_info_parts(plot_df_new, info)
    _price_txt.set_text(new_parts["price"])
    _change_txt.set_text(new_parts["change"])
    _change_txt.set_color(new_parts["change_color"])
    _meta_txt.set_text(new_parts["meta"])
```

> **設計根拠:** 銘柄名（`name`）は更新データでも変わらないため `animate()` 内での更新は不要。`_price_txt` / `_change_txt` / `_meta_txt` のみ更新する。`fig.text()` オブジェクトはフィギュアレベルのアーティストであり `ax.clear()` で消えない。
