## Findings Summary

### Yahoo!ファイナンス Japan のデータ取得方法

#### 1. 公式API
- **公式API は存在しない。** `finance.yahoo.co.jp` は開発者向けAPIを公開していない。

#### 2. yfinance ライブラリ（プロトタイプ・個人利用向け）
- 東証銘柄は `.T` サフィックスで指定: `7203.T` (トヨタ)、`6758.T` (ソニー) など
- `yf.Ticker("7203.T").info` で銘柄情報を取得可能
- 取得できる主なフィールド:
  - 銘柄名、ティッカー、市場（東証PRM等）
  - 現在値、前日終値、日中変化額・変化率、始値、高値、安値
  - 出来高、売買代金
  - 時価総額、PER、PBR、EPS、BPS
  - 配当利回り、年間配当額
  - 52週高値・安値
- `finance.yahoo.co.jp` の銘柄ページURL: `https://finance.yahoo.co.jp/quote/{銘柄コード}.T`

#### 3. J-Quants API（JPX公式・本番利用向け）
- JPX（日本取引所グループ）が運営する公式データAPI
- 2025年12月にV2リリース（APIキー認証方式に変更）
- Freeプランあり（12週遅延データ）
- Python クライアント: `jquants-api-client`
- 2026年1月更新でCSV配信・分足・ティックデータ追加

## Risks / Constraints Discovered

- **ToS違反リスク:** Yahoo! の利用規約は自動クロール・スクレイピングを明示的に禁止している。`yfinance` の利用は技術的にToS違反に該当する可能性がある。個人・研究用途では執行されることは稀だが、商用利用には適さない。
- **URL構造の変更リスク:** Yahoo!ファイナンスのHTML構造や内部エンドポイントは予告なく変更される可能性があり、`yfinance` が突然動作しなくなるリスクがある。
- **銘柄コード体系の変更:** 2025年の市場区分再編（SICC改革）により、一部銘柄コードが英数字混在に変わっている場合がある。サフィックス `.T` の扱いを要確認。
- **レート制限:** 短時間に大量リクエストを行うとIPブロックされる可能性がある。
- **認証・APIキー管理:** J-Quants API を選択する場合、APIキーの安全な管理が必要。
