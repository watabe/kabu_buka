# チャート常にCUIで取得した会社情報も全て表示する

- **Date:** 2026-02-21

## Why Needed

現在表示しているグラフがなんなのかを一目でわかるようにしたい。また何日何時現在の情報なのかもわかるようにしたい。

現状、`--chart` フラグでローソク足チャートを表示しても、会社名・株価・出来高などの CUI 出力はターミナルに残るだけであり、グラフウィンドウ単体では銘柄・取得日時・基本指標が一切わからない。チャートを見ながら CUI 画面を確認する手間を解消し、グラフだけで文脈が完結するようにする。

---

## Scope

### In-scope
- `--chart` フラグ使用時、ローソク足チャートに CUI で取得した会社情報（銘柄コード・会社名・現在値・前日比・出来高・時価総額など）を追加表示する
- 取得日時（何日何時現在）をチャートに明示する
- `plot_intraday()` の引数として `StockInfo` を受け取るよう拡張する（または補助情報を dict で受け取る）
- `mplfinance` の `fig.text()` / `axes.set_title()` / `figtext()` など既存 API を活用して実現する

### Out-of-scope
- CUI のテキスト出力フォーマットの変更
- リアルタイム更新・自動リフレッシュ
- チャートのレイアウト・配色の全面見直し
- `--chart` なしの通常起動時の動作変更

---

## Success Criteria

- チャートウィンドウ内に銘柄コード・会社名が明示されている
- 取得日時（日付・時刻）がチャート内に表示されている
- 現在値・前日比・出来高など CUI と同等の主要指標が1箇所以上チャート内に表示されている
- `plot_intraday()` の既存テストがすべて PASS のまま維持される
- 新規テストが追加され、追加情報の表示を検証している

---

## Risks / Unknowns

- `mplfinance` のレイアウト API（`fig.text` / `plt.figtext`）がバージョン間で挙動が異なる可能性がある
- `mpf.plot()` は `fig` オブジェクトを直接返さない（`returnfig=True` オプションが必要）。現在のコードが `returnfig` を使用していないため、改修が必要
- 会社情報テキストの配置・フォントサイズがチャートサイズ（14×7 inch）に対して適切かどうかは実際に描画して確認が必要
- `plot_intraday(df, ticker)` シグネチャ変更により、既存の呼び出し側（`main.py`）の変更も必要になる
