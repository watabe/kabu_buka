# 株価情報 CLI

Yahoo! Finance から東証銘柄の株価情報を取得し、CUI 表示・ローソク足チャート表示・テクニカルパターン予測を行うコマンドラインツールです。

---

## 機能一覧

| 機能 | 説明 |
|---|---|
| 株価情報表示 | 銘柄名・現在値・前日比・出来高・時価総額を CUI に出力 |
| チャート表示 | 当日1分足のローソク足チャートを GUI ウィンドウで表示 |
| 自動更新 | 指定インターバル（秒）ごとにチャートをリアルタイム更新 |
| パターン予測 | テクニカルパターンを自動検出し、将来予測ラインをチャートにオーバーレイ表示 |

---

## 必要環境

- Python 3.12 以上
- GUI 表示に対応した環境（macOS / Windows / Linux + X11）

### 依存ライブラリ

```
yfinance>=1.2.0
mplfinance>=0.12.10
numpy
pandas
matplotlib
```

インストール:

```bash
pip install -r requirements.txt
```

> チャート表示が失敗する場合は `matplotlib` のバックエンドを明示してください:
> ```bash
> MPLBACKEND=TkAgg python main.py 7203 --chart
> ```

---

## 使い方

### 基本構文

```
python main.py <銘柄コード> [--chart] [--interval SEC] [--predict]
```

### 引数・オプション

| 引数/オプション | 必須 | 説明 |
|---|---|---|
| `銘柄コード` | ✓ | 東証銘柄コード（例: `7203`） |
| `--chart` | | ローソク足チャートを GUI ウィンドウで表示する |
| `--interval SEC` | | チャートを N 秒ごとに自動更新する（`--chart` と併用） |
| `--predict` | | チャートパターンを検出して将来予測ラインをオーバーレイ表示する |

---

## 実行例

### 株価情報を CUI で表示

```bash
python main.py 7203
```

```
トヨタ自動車 (7203.T)
現在値  ¥3,410   +18 (+0.53%)
始値    ¥3,400   高値  ¥3,420   安値  ¥3,390
出来高  560,000  時価総額  ¥55.0兆
```

---

### ローソク足チャートを表示

```bash
python main.py 7203 --chart
```

当日1分足の OHLCV データをローソク足チャートで表示します。チャート上部に銘柄名・現在値・前日比・出来高・時価総額を表示します。

---

### チャートを自動更新

```bash
python main.py 7203 --chart --interval 60
```

60 秒ごとにデータを再取得してチャートを更新し続けます。ウィンドウを閉じるか `Ctrl+C` で停止します。

---

### チャートにパターン予測を表示

```bash
python main.py 7203 --chart --predict
```

以下のテクニカルパターンを自動検出し、将来 10 本分の予測ラインをチャートに橙色の点線でオーバーレイ表示します。

| パターン名 | 方向 | 検出条件 |
|---|---|---|
| ヘッドアンドショルダー | 下降 | 極大 3 つ・中央が最高・両肩の価格差 ≤ 3% |
| 逆ヘッドアンドショルダー | 上昇 | 極小 3 つ・中央が最低・両肩の価格差 ≤ 3% |
| ダブルトップ | 下降 | 極大 2 つ・価格差 ≤ 2%・間に極小が存在 |
| ダブルボトム | 上昇 | 極小 2 つ・価格差 ≤ 2%・間に極大が存在 |

- データが 30 本未満の場合はパターン検出をスキップし「パターン未検出」と表示します。
- 検出されたパターン名はチャート内に表示されます。

---

### 自動更新 + パターン予測

```bash
python main.py 7203 --chart --interval 60 --predict
```

データ更新のたびにパターンを再検出し、予測ラインを再描画します。

指定した銘柄コードの株価情報を Yahoo! Finance から取得して表示します。

## エラーと対処法

| エラーメッセージ | 原因 | 対処法 |
|---|---|---|
| `銘柄が見つかりませんでした: XXXX` | 銘柄コードが存在しない | 正しい東証銘柄コードを指定 |
| `データ取得に失敗しました。通信状況を確認してください。` | ネットワーク障害 / タイムアウト | インターネット接続を確認 |
| `当日の取引データがありません: XXXX` | 取引時間外または休場日 | 取引時間中（9:00〜15:30）に実行 |
| `グラフ表示に失敗しました。環境変数 MPLBACKEND=TkAgg を設定して再実行してください。` | GUI バックエンド未設定 | `MPLBACKEND=TkAgg python main.py ...` で実行 |
